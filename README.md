# 🚗 Luxury Wheels Braga

Site de aluguer de carros desenvolvido com Python e Flask.

## 📋 Funcionalidades
- Registo e autenticação de utilizadores
- Catálogo de carros com fotos
- Reserva de carros com escolha de datas e locais
- Cálculo automático do preço por dias
- Página de pagamento
- Gestão de reservas (editar e cancelar)
- Página de contacto

## 🛠️ Tecnologias utilizadas
- Python
- Flask
- SQLite + Flask-SQLAlchemy
- HTML5 + CSS3
- Jinja2

## 🚀 Como instalar e executar
1. Clona o repositório
2. Instala as dependências: `pip install flask flask-sqlalchemy`
3. Executa: `python app.py`
4. Abre o browser em `http://127.0.0.1:5000`
